## 🎯 개요

**자금왕**은 영업직 종사자를 위한 기업회원 관리 및 자금 추천 플랫폼입니다. 고객사 정보를 체계적으로 관리하고, AI 기반 자금 추천을 통해 영업 효율을 극대화할 수 있는 올인원 솔루션을 제공합니다.

## 💡 기획 배경

### 문제 인식
- 영업직은 다수의 기업 고객을 관리하며 각 기업에 맞는 자금 정보를 제공해야 하는 부담
- 엑셀, 메모장 등 분산된 도구로 고객 정보를 관리하여 비효율 발생
- 적시에 고객에게 적합한 지원사업이나 자금 정보를 제공하기 어려움
- 고객별 상담 내역과 파일을 체계적으로 관리할 방법 부족

### 솔루션
- **통합 고객 관리**: 한 곳에서 모든 기업회원 정보를 체계적으로 관리
- **AI 기반 자금 추천**: 기업 정보를 분석하여 맞춤형 자금/지원사업 자동 추천
- **스마트 노트 기능**: 고객별 메모와 파일을 저장하여 상담 이력 관리
- **자동 알림 시스템**: 구독 및 사용 기간 만료 전 카카오톡 알림
- **간편 결제**: 토스 페이먼츠를 통한 이용권 구매

## ✨ 주요 기능

### 1. 기업회원 정보 관리
- 다수의 기업 고객 정보 통합 저장 및 관리
- 기업별 상세 프로필 (업종, 업력, 규모, 지역 등)
- 직관적인 대시보드로 고객 현황 한눈에 파악
- 검색 및 필터링으로 빠른 고객 정보 조회

### 2. AI 기반 맞춤형 자금 추천
- 기업 정보 분석을 통한 적합한 지원사업 추천
- OpenAI GPT를 활용한 지능형 매칭
- 기업마당 API 연동으로 최신 지원사업 정보 제공
- 추천 결과를 고객에게 바로 공유 가능

### 3. 스마트 노트 기능
- 기업별 상담 메모 작성 및 저장
- 관련 파일(계약서, 서류 등) 첨부 및 관리
- 시간순 정렬로 상담 이력 추적
- AWS S3 기반 안전한 파일 보관

### 4. 구독 및 결제 시스템
- 토스 페이먼츠 연동 간편 결제
- 이용권 기반 구독 모델
- 자동 갱신 및 결제 내역 관리
- 다양한 결제 수단 지원

### 5. 자동 알림 서비스
- SOLAPI 카카오톡 알림톡 연동
- 구독 만료 사전 알림
- 사용 기간 종료 안내
- 중요 공지사항 푸시

### 6. 회원 인증 시스템
- 이메일 인증을 통한 안전한 회원가입
- 카카오 소셜 로그인 지원
- Google Email API를 활용한 인증 메일 발송
- 비밀번호 재설정 기능

### 7. OCR 및 음성 지원
- Naver CLOVA OCR로 문서 자동 인식
- Naver CLOVA STT로 음성 메모 텍스트 변환
- 다양한 형식의 문서 처리

## 🛠 기술 스택

### Backend
- **Language**: Python
- **Framework**: Django
- **Database**: MySQL

### Frontend
- **Language**: JavaScript

### Infrastructure
- **Deployment**: AWS EC2 (Ubuntu)
- **Web Server**: Nginx
- **WSGI Server**: Gunicorn
- **Storage**: AWS S3

### Authentication
- **Framework**: django-allauth
- **Social Login**: Kakao OAuth 2.0
- **Email Verification**: Google Email API

### Payment & Notification
- **Payment Gateway**: Toss Payments API
- **Messaging**: SOLAPI (카카오톡 알림톡)

### External APIs
- **OpenAI API**: GPT 모델을 활용한 자금 추천 및 분석
- **Naver CLOVA API**: 
  - OCR (문서 텍스트 추출)
  - STT (음성-텍스트 변환)
- **Bizinfo API**: 기업마당 API를 통한 지원사업 데이터 수집
- **SOLAPI**: 카카오톡 알림톡 발송
- **Toss Payments API**: 결제 처리
- **Google Email API**: 인증 메일 발송

## 🔑 핵심 기능 상세

### 1. 통합 기업회원 관리 시스템
```
고객 정보 입력 → MySQL 저장 → 대시보드 표시
→ 검색/필터링 → 상세 정보 조회 → 수정/삭제
```

**주요 관리 항목**
- 기업 기본 정보 (상호명, 사업자번호, 대표자 등)
- 업종 및 업력
- 직원 수, 매출 규모
- 지역 정보
- 연락처 및 담당자 정보

### 2. AI 기반 자금 추천 엔진
```
기업 정보 분석 → OpenAI GPT 처리 
→ Bizinfo API 매칭 → 추천 결과 생성 
→ 영업자에게 제공
```

**추천 알고리즘**
- 기업 특성 분석 (업종, 규모, 지역)
- 지원사업 요건 매칭
- AI 기반 적합도 점수 산출
- 우선순위별 추천 리스트 제공

### 3. 스마트 노트 시스템
```
메모 작성 → 텍스트 저장 (MySQL)
파일 첨부 → S3 업로드 → URL 저장
음성 메모 → Naver CLOVA STT → 텍스트 변환
문서 스캔 → Naver CLOVA OCR → 텍스트 추출
```

**기능 특징**
- 파일 형식 제한 없음 (문서, 이미지, PDF 등)
- 시간순/고객별 정렬
- 태그 기능으로 분류 관리

### 4. 구독 관리 및 결제
```
이용권 선택 → Toss Payments 호출 → 결제 진행
→ 결제 완료 → 구독 활성화 → DB 업데이트
→ 만료일 계산 → 자동 알림 예약
```

**구독 플로우**
- 다양한 이용권 옵션 제공
- 자동 결제 갱신 지원
- 결제 내역 조회 및 영수증 발행
- 환불 처리 시스템

### 5. 카카오톡 알림 시스템
```
만료일 체크 (일일 배치) → 알림 대상 선별
→ SOLAPI API 호출 → 카카오톡 알림톡 발송
→ 발송 이력 저장
```

**알림 종류**
- 구독 만료 7일 전 사전 알림
- 구독 만료 1일 전 최종 알림
- 결제 완료 확인
- 서비스 업데이트 공지

### 6. 이메일 인증 시스템
```
회원가입 → 이메일 입력 → Google Email API 호출
→ 인증 메일 발송 → 링크 클릭 → 인증 완료
→ 계정 활성화
```

**보안 기능**
- 이메일 중복 확인
- 인증 토큰 생성 (시간 제한)
- 안전한 비밀번호 정책
- 소셜 로그인 연동

### 7. 멀티미디어 처리
```
문서 업로드 → 파일 타입 확인
→ 이미지/PDF: CLOVA OCR → 텍스트 추출
→ 음성 파일: CLOVA STT → 텍스트 변환
→ 노트에 자동 첨부
```
